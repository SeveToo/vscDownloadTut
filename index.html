Pobieranie

Wchodzimy na stronę Visual Studio Code w zakładkę download
Visual Studio Code Download

Jeśli nie wiesz ilu bitowy jest twój system windows:

Klikasz prawym przyciskiem myszy na Ten Komputer

Klikamy Właściwości

Teraz wystarczy kliknąć odpowiednią wersję np:

Czekamy...

Instalacja

Następnie otwieramy pobrany plik

Wybieramy język

Akceptujemy postanowienia licencyjne

Zaznaczamy check box Create a desktop icon

Klikamy Install

Instalacja zakończona, teraz uruchamiamy Visual Studio Code, który jeśli
postępowałęś/łaś według powyższych instrukcji to skrót powinien być na pulpicie

Konfiguracja

Klikamy Extensions lub Ctrl+Shift+x

Instalujemy rozszerzenia Live Share i Live Serwer klikając Install

Po zainstalowaniu rozszerzeń najlepiej jest zrestartować program

Następnie klikamy LiveShare, co spowoduje otwarcie przeglądarki, w której będziemy musieli się zalogować.

Logowanie

W tym miejscu wybieramy metodę logowania

W moim przypadku to będzie GitHub na, którym się loguję

Teraz klikamy Authorize VisualStudioLiveShare

Powinniśmy zobaczyć takie okno potwierdzające pomyślne połączenie naszego konta z usługą LiveShare

w Visual Studio Code powinniśmy zobaczyć swój NICK
